# Disaster Damage Assessment Datasets

This project uses publicly available datasets related to **disaster response, aerial imagery, and building damage assessment**.

## 1. DIUx xView 2018 Detection Challenge
- **Name:** xView Dataset  
- **Description:** Large-scale object detection dataset using high-resolution satellite imagery. It contains annotated objects such as buildings, vehicles, and infrastructure useful for disaster and urban analysis.  
- **Link:** https://challenge.xviewdataset.org/

---

## 2. xView2: Assess Building Damage
- **Name:** xView2 Dataset  
- **Description:** Dataset designed for **post-disaster building damage assessment** using satellite imagery. Includes **pre-disaster and post-disaster images** with damage labels (no damage, minor, major, destroyed).  
- **Link:** https://xview2.org/

---

## 3. AIDER: Aerial Image Database for Emergency Response
- **Name:** AIDER Dataset  
- **Description:** A dataset containing aerial images for **emergency response applications**, supporting tasks such as disaster monitoring and scene understanding.  
- **Repository:** https://github.com/ckyrkou/AIDER

---

## 4. ISBDA Dataset
- **Name:** ISBDA (Image-Based Structural Damage Assessment)  
- **File:** `ISBDA-ISBDA.zip`  
- **Description:** Dataset used for structural damage detection in buildings after disasters using aerial or satellite imagery.

---

## 5. FloodNet Dataset
- **Name:** FloodNet  
- **Description:** Dataset designed for **flood scene understanding**, containing annotated flood images for segmentation and classification tasks.  
- **Repository:** https://github.com/BinaLab/FloodNet-Challenge-EARTHVISION2021

---

## 6. DesignSafe Data Depot
- **Name:** DesignSafe-CI Data Repository  
- **Description:** Platform providing datasets related to **natural hazards engineering**, including hurricanes, floods, earthquakes, and structural damage.  
- **Link:** https://www.designsafe-ci.org/data/

---

## 7. xBD Dataset
- **Name:** xBD (Extreme Building Damage Dataset)  
- **Folder/File:** `xbd_dataset`  
- **Description:** One of the **largest datasets for building damage assessment** using satellite imagery, covering multiple disasters such as earthquakes, wildfires, and hurricanes.  
- **Link:** https://xview2.org/dataset
